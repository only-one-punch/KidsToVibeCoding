

# CodeBuddyAI 页面设计稿 (Cinematic v1.2)

**设计哲学**: 动力学教育美学 (The Kinetic Education Aesthetic)
**核心技术栈**: React, Vite, Tailwind CSS, shadcn/ui, Framer Motion
**设计参考**: Velorah® 极简电影风格

---

## 0. 全局样式定义 (CSS Variables)

按照你的提示词要求，所有页面遵循以下视觉底座：

*   **背景**: `hsl(201 100% 13%)` (深邃海军蓝) + 全屏循环视频
*   **字体**:
    *   `--font-display`: 'Instrument Serif', serif (用于标题、Logo)
    *   `--font-body`: 'Inter', sans-serif (用于正文、数据)
*   **特效**: `.liquid-glass` (1.4px 渐变描边 + 4px 模糊)
*   **动效**: `animate-fade-rise` (24px 向上位移 + 0.8s 渐进)

## 设计规范摘要 (Standard Rules)

| 维度 | 规则 |
| :--- | :--- |
| **色彩** | 仅限 HSL(201 100% 13%) 背景，不允许出现亮彩色块。 |
| **按钮** | 必须应用 `.liquid-glass` 样式，边框厚度 1.4px，圆角圆润。 |
| **排版** | 核心标题间距 `-2.46px`。重要词汇必须用 `em` 降噪处理。 |
| **动效** | 页面加载必须逐级向上浮现，杜绝瞬间刷新。 |
| **留白** | 保持“画廊”般的空间感，禁止内容拥挤。 |
